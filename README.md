# Obsidian Budget Planner Plugin

Welcome to the Obsidian Budget Planner Plugin! This plugin provides an easy way to planing your budget within 
Obsidian using a markdown code block named budget.

## Examples:

### Without category

```budget
Spotify    | $4.99
Youtube    | $16.99
1Password  | 6.95
```

### With category

```budget
Online services:
	Spotify    | $4.99
	Youtube    | $16.99
	1Password  | 6.95
Others:
	Netflix    | $12.99
```
